Free-friendly default

Gemini

Best first choice if you want a free Google API key and simple setup.

  1. Open Google AI Studio.
  2. Sign in with a Google account.
  3. Create a Gemini API key.
  4. Paste it into this app and choose Save key.

High quality

OpenAI API

Best when you want fast server-side summaries through your OpenAI API account.

  1. Open the OpenAI API keys page.
  2. Create a new project key.
  3. Paste it into this app and choose Save key.
Get OpenAI key OpenAI docs

Model router

OpenRouter API

Best when you want one API key that can route to different model providers.

  1. Open OpenRouter settings.
  2. Create an API key.
  3. Paste it into this app and choose Save key.
Get OpenRouter key OpenRouter docs

Local login

OpenAI Login

Best for local or private deployments where the Codex CLI can keep an OpenAI login session.

  1. Install the Codex CLI in the environment running the app.
  2. Choose OpenAI Login in the app.
  3. Complete the OAuth popup.
Codex CLI docs

Deployment keys

For Docker or server deployments, prefer environment variables or Docker secrets. The app also supports saving keys to the server-side data volume so users do not need to retype them.

GEMINI_API_KEY=...
OPENAI_API_KEY=...
OPENROUTER_API_KEY=...
